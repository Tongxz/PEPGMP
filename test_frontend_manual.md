## 🔧 测试工具和方法

### 浏览器开发者工具
1. **控制台 (Console)**
   - 检查JavaScript错误
   - 查看API请求和响应
   - 监控性能指标

2. **网络 (Network)**
   - 监控API请求状态
   - 检查资源加载时间
   - 分析网络性能

3. **元素 (Elements)**
   - 检查HTML结构
   - 调试CSS样式
   - 测试响应式设计

4. **性能 (Performance)**
   - 分析页面加载性能
   - 检查内存使用情况
   - 优化建议

### 手动测试步骤

1. **准备环境**
   ```bash
   # 确保后端API服务运行
   python main.py --mode api --port 8000 --debug

   # 确保前端开发服务器运行
   cd frontend && npm run dev
   ```

2. **打开浏览器**
   - 访问 `http://localhost:5173/`
   - 打开开发者工具 (F12)
   - 清除缓存和Cookie

3. **逐项测试**
   - 按照上述清单逐项测试
   - 记录发现的问题
   - 截图保存证据

4. **性能测试**


## 🚀 自动化测试

如果需要自动化测试，可以使用提供的测试脚本：

```bash
# 安装依赖
npm install puppeteer

# 运行自动化测试
node test_frontend_functionality.js
```

自动化测试将覆盖：
- 页面导航测试
- UI交互测试
- API连接测试
- 性能基准测试
