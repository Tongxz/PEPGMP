# 搜索功能移除记录

## 📅 移除日期
2025-11-04

## 🎯 移除原因
1. 系统规模小（3个摄像头），不需要全局搜索
2. 各页面已有完善的筛选功能
3. 搜索功能导致系统出现错误
4. 核心功能是人体行为检测，应专注于此

## ✅ 已移除的文件

### 前端文件
- ✅ `frontend/src/components/common/GlobalSearch.vue` - 全局搜索组件
- ✅ `frontend/src/views/Search.vue` - 搜索页面
- ✅ `frontend/src/api/search.ts` - 搜索API客户端

### 后端文件
- ✅ `src/api/routers/search.py` - 搜索API路由

## 🔧 已修改的文件

### 前端
1. **`frontend/src/layouts/MainLayout.vue`**
   - 移除 `GlobalSearch` 组件导入
   - 移除头部搜索组件使用

2. **`frontend/src/router/index.ts`**
   - 移除 `/search` 路由定义

### 后端
1. **`src/api/app.py`**
   - 移除 `search` 模块导入
   - 移除搜索路由注册

## 🎯 替代方案

### 如果用户需要查找内容

1. **查找摄像头**
   - 使用"相机配置"页面的列表和筛选功能

2. **查找检测记录**
   - 使用"历史记录"页面的摄像头筛选和时间范围筛选

3. **查找违规事件**
   - 使用"历史记录"页面的违规记录筛选（类型、状态）

4. **查找告警**
   - 使用"告警中心"页面的列表展示

## 📝 后续建议

如果将来系统规模扩大（超过20个摄像头或数据量显著增加），可以考虑：
1. 重新评估是否需要搜索功能
2. 或者采用页面级搜索（更轻量级）
3. 或者添加简单的ID搜索输入框
